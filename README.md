# BSynt - Command & Syntax Documentation

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)
[![React](https://img.shields.io/badge/React-18-blue.svg)](https://reactjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5-blue.svg)](https://www.typescriptlang.org/)

Dokumentasi lengkap syntax dan perintah untuk **Linux, Windows, macOS, MySQL, PostgreSQL, Git, Docker, Redis, dan Penetration Testing** dengan penjelasan detail, opsi lengkap, dan contoh penggunaan praktis.

🌐 **Live Demo**: [https://bsynt.beznproject.web.id](https://bsynt.beznproject.web.id)

## ✨ Features

- 📚 **129+ Halaman Dokumentasi** - 9 kategori teknologi
- 🔍 **Global Search** - Cari command dengan cepat
- 📱 **Responsive Design** - Mobile-friendly
- 🌓 **Dark/Light Mode** - Theme toggle
- 📑 **Dynamic TOC** - Table of contents berubah per kategori
- 🔗 **Share Button** - Bagikan ke WhatsApp, Telegram, Facebook, Twitter
- ⚡ **Fast Performance** - Built with Vite
- 🎨 **Modern UI** - Tailwind CSS + shadcn/ui

## 📋 Dokumentasi Tersedia

### Operating Systems
- **Linux** (19 topik) - Navigasi, File Management, Text Processing, System Management
- **Windows** (19 topik) - CMD & PowerShell commands
- **macOS** (15 topik) - Terminal commands & utilities

### Databases
- **MySQL** (20 topik) - Database, Tables, CRUD, Joins, Functions, Triggers
- **PostgreSQL** (20 topik) - Similar to MySQL + PostgreSQL specific features

### Development Tools
- **Git** (13 topik) - Version control, branching, merging
- **Docker** (6 topik) - Container, Image, Volume, Network, Compose
- **Redis** (7 topik) - Data types, Keys management

### Security
- **Penetration Testing** (9 topik) - Nmap, Metasploit, Burp Suite, OSINT, Password Attacks

## 🚀 Quick Start

### Prerequisites

- Node.js 18+ 
- npm or bun

### Installation

```bash
# Clone repository
git clone https://github.com/nafiurohman/bsynt.git
cd bsynt

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🛠️ Tech Stack

- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **State Management**: TanStack Query

## 📦 Build for Production

```bash
# Build
npm run build

# Preview production build
npm run preview
```

Output akan ada di folder `dist/`

## 🌐 Deployment

### Vercel (Recommended)

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```

Atau import repository di [vercel.com](https://vercel.com)

### Build Settings
- Framework Preset: **Vite**
- Build Command: `npm run build`
- Output Directory: `dist`
